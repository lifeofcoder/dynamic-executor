# dynamic-executor
在线动态执行器，可以在线上环境直接编译执行任意代码。
## 功能介绍
该项目基于”深入理解Java虚拟机“中提到的原理，实现了这个在线动态执行器。其能够将任意代码直接发送到线上编译并执行。且能够返回最终结果。

## 用途
1.	在线打印日志信息
2.	在线上执行某段业务逻辑
3.	线上调测上游接口
4.	定位线上问题
5.	修改线上日志级别
6.	操作线上数据记录
7.	...
